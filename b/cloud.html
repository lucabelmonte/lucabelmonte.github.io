Cloud

17/10/16

La nostra prima esperienza di alternanza scuola lavoro inizia con l'argomento del 'cloud', e nella sua sempre piu' maggior importanza nelle aziende.

Cos'e' quindi il cloud? Con Cloud o Cloud Computing si indica il decentramento della potenza di calcolo e di storage di qualunque informazione, mediante l'utilizzo internet.

Le principali aziende che offrono questo servizio sono:

  • Amazon Web services
  • Microsoft
  • Google

NodeJs

Dopo un'introduzione al Cloud Computing abbiamo iniziato a lavorare con NodeJs: e' un derivato del JavaScript per le applicazioni lato server. Una delle sue maggiori caratteristica e' che sfrutta i dati in modo asincrono attravarso l'utilizzo delle callback e quindi non elabora i dati uno alla volta ma in modo parallelo.

Pro:

  • Asincrono
  • Utilizzo del javascript sia lato server che lato client
  • Asincrono

Contro

  • Necessita molta potenza di calcolo
  • Difficile uso con database
  • Poco performante per lavori CPU bound
